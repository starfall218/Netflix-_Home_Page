🎬 Netflix Clone (Frontend Only)

A Netflix-inspired web application featuring user authentication screens and personalized UI experiences based on user input.

---

🚀 Features

- 🔐 Authentication UI

   - Sign-up page
   - Login page
(Frontend only — no real authentication or database)

- 🎯 Personalized Screens

   - UI adapts based on user-entered data
   - Simulated user experience without backend logic

- 🎨 Clean & Responsive Design

   - Modern Netflix-style layout
   - Works across different screen sizes

- ⚡ Dynamic Frontend Behavior

   - Conditional rendering based on input
   - Smooth navigation between pages

---

🛠️ Tech Stack

- React.js
- CSS


---

⚠️ Limitations

- No backend or database
- No real user authentication
- Data is not stored or persisted
- Personalization is simulated on the frontend

---

💡 Purpose

This project was built to:

- Practice frontend development skills
- Understand UI/UX design patterns
- Simulate authentication flows
- Explore dynamic rendering without a backend
